# cisco 三层交换机实现vlan间通信
Cisco二层交换机是不能实现vlan间通讯的，这就需要三层路由功能，而Cisco3560交换机默认情况下是二层交换机，这就需要对交换机进行一些配置，实现vlan间的通信。
